# Tic-Tac-Toe

A professional, browser-based Tic-Tac-Toe game built with pure HTML, CSS, and JavaScript — no frameworks, no build step.

## Play

Open `index.html` directly in any modern browser.

## Features

- **Score tracking** — X wins, O wins, and draws persist across rounds
- **Win highlight** — winning cells glow red (X) or cyan (O) with a pulse animation
- **Animations** — marks bounce in on play, score bumps on increment, result slides up
- **Result overlay** — modal announces the winner with a Play Again button
- **New Game** — resets the board while keeping scores
- **Reset Scores** — zeroes all counters
- **Dark theme** — deep navy background with vivid player colors
- **Responsive** — adapts to mobile screens

## Project Structure

```
tic-tac-toe/
├── index.html   # Markup and layout
├── style.css    # Design tokens, animations, responsive styles
└── game.js      # Game state, logic, and DOM rendering
```

## Tech Stack

- HTML5 (semantic, accessible)
- CSS3 (custom properties, grid, keyframe animations)
- Vanilla JavaScript (ES6+, no dependencies)
