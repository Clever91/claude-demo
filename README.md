# claude-demo

A learning project for exploring [Claude Code](https://claude.ai/code) — Anthropic's AI-powered CLI for software engineering.

## Projects

### [Tic-Tac-Toe](./tic-tac-toe/)
A professional, browser-based Tic-Tac-Toe game built with pure HTML, CSS, and JavaScript.

**Features:** score tracking, win animations, result overlay, responsive dark theme.
**Run:** open `tic-tac-toe/index.html` in any browser — no server needed.

## Purpose

This repo documents hands-on experimentation with Claude Code features:
- Creating and structuring projects
- Git workflow (branches, commits, pull requests)
- GitHub integration
- CLAUDE.md project context
- Memory and settings configuration

## Tech Stack

All projects in this repo use plain web technologies (HTML/CSS/JS) with no frameworks or build tools unless otherwise noted.
